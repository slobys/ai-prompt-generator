# ai-prompt-generator
🌈 AI绘画标签生成器：将标签进行分类整理，支持中文显示、加权、降权、成人模式等功能，最讨厌未经同意将其设置成各种收费版本。

## 使用方法

> 下载该库
```
https://github.com/slobys/ai-prompt-generator.git
```

> 安装依赖
```
npm install
```

> 运行预览
```
npm run serve
```
